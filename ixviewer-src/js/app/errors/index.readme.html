Functions

  • Errors.checkPerformanceConcern()
    • Purpose:
      • Gives User warning if Total Fact Count is above 7500 if User is using Chrome browser, or if Total Fact Count is above 1000 if User is NOT using Chrome browser.
    • Arguments:
      • totalTaxonomies: Number
  • Errors.checkFileSizeForLimits()
    • Purpose:
      • Gives User Warning if XHTML Form size is greater than Constants.fileSizeError[1]
    • Arguments:
      • fileSize: Number
  • Errors.updateMainContainerHeight()
    • Purpose:
      • Finds total number of children within #error-container which refers to the amount of warnings and errors being shown to the user. Updates the height of .main-container to ensure the height of the application stays consistent.
    • Arguments:
      • removeWarning: Boolean
