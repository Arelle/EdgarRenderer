3rd Party Libraries README

Purpose

High level Developer information regarding 3rd party Libraries used within the IXViewer

Location

All 3rd party libraries are located @ {{PROJECT ROOT}}/js/lib

List

  • Font Awesome 5.5.0
    • Purpose: Font icons
  • Bootstrap 4.1.3
    • Purpose: Responsive Utilities, HTML Components
  • Core-JS 3.2.1
    • Purpose: Polyfills for browsers that are not up to date
  • JQuery-Slim 3.7.1
    • Purpose: Bootstrap requires the use of jQuery
  • Moment 2.24.0
    • Purpose: Used for transformations that require dates
  • Vanilla Picker 2.10.0
    • Purpose: User Settings to change fact colors
  • Popper 1.15.0
    • Purpose: Bootstrap requires the use of popper
