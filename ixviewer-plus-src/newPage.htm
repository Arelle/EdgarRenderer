    The Inline XBRL viewer allows a user\r\n to quickly and easily review details of the tagged information in\r\n an Inline document by automatically placing a top and bottom\r\n highlight border around each tagged numeric fact and left and\r\n right border for each block tagged fact. Hovering over a tagged\r\n fact will highlight (shade) all content related to the tagged\r\n fact, and clicking on a tagged fact will reveal its tagging\r\n details in the Fact Review Window. Navigation, search and filter\r\n options are also provided to easily refine and identify specific\r\n types of tagged information.

    \r\n
    \r\n
    \r\n
    \r\n
    \r\n
    \r\n \r\n
    \r\n
    \r\n
    \r\n

    The Fact Review Window shows the tagging\r\n details for the currently selected fact, which is highlighted\r\n with a solid blue border. There are four categories of fact detail\r\n which can be viewed; an \"N/A\" value indicates there is no\r\n available information for the item within the given category:

    \r\n
      \r\n
    • Attributes - All primary information (as\r\n applicable) describing the tagged fact including period, sign,\r\n decimals, dimensional detail (axes and members), scale, measure\r\n and data type.
      • \r\n
    • Labels - Detailed documentation\r\n (definition) for the tag used, and other labels.
      • \r\n
    • References - Authoritative reference\r\n information (as applicable) for the selected tag.
      • \r\n
    • Calculation - Balance and calculation\r\n hierarchy details (numeric non-dimensioned items only).
      • \r\n
    \r\n
    \r\n
    \r\n
    \r\n
    \r\n
    \r\n \r\n
    \r\n
    \r\n
    \r\n

    There are two ways to search the\r\n document: FACTS AND SECTIONS. To search information, enter a\r\n keyword in a search box and select the magnifying glass icon to\r\n return matching results. Search operators \"and\" (via \"AND\" or\r\n \"&\") and \"or\" (via \"OR\" or \"|\") are available to further refine a\r\n search.

    \r\n

    \r\n FACTS\r\n

    \r\n

    The Search Facts box can be used to find\r\n tagged facts matching entered keywords. By default, tag name, tag\r\n labels, and tagged content are included in Search. Tagged facts\r\n matching the search criteria are shown with a yellow-colored\r\n (default) shading on the page and appear in the Facts List.\r\n Select the \"X\" button to the left of the magnifying glass icon to\r\n clear the search.

    \r\n

    The content included in Search can be increased to\r\n included tag definitions, dimensions, and authoritative\r\n references by selecting the cog / wheel icon to the left of the\r\n search box. See Settings for more information.

    \r\n\r\n

    \r\n SECTIONS\r\n

    \r\n

    The Search Sections box can be used to\r\n filter the tagged sections of the financial statements.

    \r\n
    \r\n
    \r\n
    \r\n
    \r\n
    \r\n \r\n
    \r\n
    \r\n
    \r\n

    Filters change the number of highlighted\r\n facts and items in the fact list by providing several ways to\r\n review and pinpoint certain tagged information. Multiple filters\r\n can be used at once.

    \r\n

    \r\n DATA FILTERS\r\n

    \r\n

    These filter options allow the user to\r\n refine the highlighted tagged facts by data type:

    \r\n
      \r\n
    • All - Displays all tagged data (default)
      • \r\n
    • Amounts Only - Numeric facts only
      • \r\n
    • Text Only - Textual facts only
      • \r\n
    • Calculations Only - Numeric facts\r\n participating in a calculation only
      • \r\n
    • Negatives Only - Negative Numeric amounts\r\n only
      • \r\n
    • Additional Items Only - Tagged facts\r\n without a corresponding HTML presentation (i.e., hidden facts)\r\n only
      • \r\n
    \r\n

    \r\n TAGS FILTER\r\n

    \r\n

    These filters allow the user to refine\r\n the highlighted facts by tag type:

    \r\n
      \r\n
    • All - Displays all tagged data (default)
      • \r\n
    • Standard Only- Facts with standard tags\r\n from a common fact (e.g., US_GAAP, IFRS, DEI).
      • \r\n
    • Custom Only - Facts with extension\r\n \"custom\" tags unique to the entity's document.
      • \r\n
    \r\n

    \r\n MORE FILTERS\r\n

    \r\n

    Additional filters that allow user to\r\n further refine the highlighted facts:

    \r\n
      \r\n
    • Periods - List of all used context years\r\n and reporting periods
      • \r\n
    • Measures - List of all used units of\r\n measure (e.g., US Dollars); as applicable
      • \r\n
    • Axis - List of all used axis\r\n (dimensions); as applicable (dimensions); as applicable
      • \r\n
    • Scale - List of all used scaled options\r\n (e.g., thousands, millions); as applicable
      • \r\n
    • Balance - Debit, credit; as applicable
      • \r\n
    \r\n

    Multiple filters work in conjunction\r\n with each other. For example, selecting the \"Amounts Only\" Data\r\n filter and \"Custom Only\" Tags filter will highlight only numeric\r\n tagged facts using custom tags and list them in the fact list.\r\n Active filters are displayed in the Filter toolbar as they are\r\n selected. Active filters can be removed individually by\r\n unchecking or selecting ALL option for each filter, or all at\r\n once via the \"Reset All Filters\" option.

    \r\n
    \r\n
    \r\n
    \r\n
    \r\n
    \r\n \r\n
    \r\n
    \r\n
    \r\n

    On the toolbar, select the \"count\" to\r\n the right of the word Facts (e.g., Facts 234) to reveal the Facts\r\n List results - a navigable listing of all currently highlighted\r\n tagged facts. The count represents the current number of facts.

    \r\n

    By default, all tagged facts are\r\n displayed in the Fact List Results. The list content and count\r\n reflects the currently highlighted facts (i.e., both Filters and\r\n Search criteria refine the list to match the highlighted tagged\r\n facts). Navigation controls are available to move through the\r\n list as well as move the current view to the corresponding\r\n highlighted fact location automatically. When a fact in the Facts\r\n Results List is selected, it will reveal the Fact Review Window.

    \r\n

    Certain letters may appear to the right\r\n of an item in the Fact Result list to indicate a certain\r\n property. If the letter \"A\" appears for a fact, it indicates the\r\n fact is additional data (i.e., hidden with potentially no\r\n corresponding HTML presentation). If the letter \"C\" appears, the\r\n fact is tagged with a custom tag. If the letter \"D\" appears, the\r\n fact is tagged with dimensional information.

    \r\n
    \r\n
    \r\n
    \r\n
    \r\n
    \r\n \r\n
    \r\n
    \r\n
    \r\n

    The SECTIONS toolbar item provides a\r\n listing of the tagged sections of the Inline document. Sections\r\n are divided into three groups:

    \r\n
      \r\n
    • Document & Entity Information
      • \r\n
    • Financial Statements
      • \r\n
    • Notes to the Financial Statements
      • \r\n
    \r\n

    By expanding a group and selecting a\r\n section item in the listing, Inline XBRL viewer will navigate to that section.\r\n When the Tagged Sections feature is open, the Search Sections box\r\n will additionally filter the list of sections to only those\r\n sections that match the entered criteria.

    \r\n
    \r\n
    \r\n
    \r\n
    \r\n
    \r\n \r\n
    \r\n
    \r\n
    \r\n

    \r\n Information\r\n

    \r\n

    The Information menu item provides\r\n additional detail about the current Inline document and\r\n customizable viewer settings.

    \r\n
      \r\n
    • Company and Document - Basic company and document\r\n information
      • \r\n
    • Tags - Fact and tag (standard and custom)\r\n information
      • \r\n
    • Files - Files used
      • \r\n
    • Additional Items - Additional hidden data\r\n that's been tagged but potentially does not have a corresponding\r\n location in the HTML
      • \r\n
    \r\n

    \r\n Save XBRL Instance\r\n

    \r\n

    The Save XBRL Instance menu item allows\r\n an XBRL instance document (*.xml) that is extracted from the\r\n Inline document to be saved locally for use by software\r\n supporting the XBRL format.

    \r\n

    \r\n Save XBRL Zip\r\n

    \r\n

    The Save XBRL Zip menu item allows a zip\r\n file (*.zip) that contains the as-provided XBRL instance document\r\n and related custom fact files to be saved locally.

    \r\n

    \r\n Settings\r\n

    \r\n

    The Settings menu item provides the\r\n ability to customize Viewer features.

    \r\n

    \r\n Tagged Fact Hover\r\n

    \r\n
      \r\n
    • On - Displays the hover fact review\r\n window for any tagged fact*
      • \r\n
    • Off - Hides the hover fact review window\r\n for any tagged fact (default)
      • \r\n
    \r\n

    * May impact performance with certain\r\n web browsers.

    \r\n

    \r\n Auto Scrolling Position (This setting will have\r\n no effect on IE 10 or Safari)\r\n

    \r\n
      \r\n
    • Top - Selecting a fact from the Sections\r\n Menu or the Fact Menu will automatically scroll that element to\r\n the Top of the Inline XBRL viewer window (default)
      • \r\n
    • Center - Selecting a fact from the\r\n Sections Menu or the Fact Menu will automatically scroll that\r\n element to the middle of the Inline XBRL viewer window
      • \r\n
    \r\n

    \r\n Highlight Colors (Use the save and reset buttons\r\n to save the selected setting or reset to default)\r\n

    \r\n
      \r\n
    • Tagged Data - Change the highlight color\r\n of the tagged fact border
      • \r\n
    • Search Results - Change the background\r\n color of tagged items matching the Search results
      • \r\n
    • Selected Fact - Change the color of\r\n highlight border used to identify the currently selected fact
      • \r\n
    • Tag Shading (hover) - Change the color of\r\n the shading applied to tagged data on hover
      • \r\n
    \r\n

    \r\n Search Options\r\n

    \r\n
      \r\n \r\n
    • Include Labels - Extends Search to\r\n include tag labels
      • \r\n
    • Include Definitions - Extends Search to\r\n include tag definitions
      • \r\n
    • Include Dimensions - Extends Search to\r\n include dimensional detail
      • \r\n
    • Include References - Extends Search to\r\n include authoritative reference information. Optionally specify\r\n the reference parts to include in the search (e.g., topic,\r\n subtopic, section)
      • \r\n

      The Inline XBRL viewer allows a user\r\n to quickly and easily review details of the tagged information in\r\n an Inline document by automatically placing a top and bottom\r\n highlight border around each tagged numeric fact and left and\r\n right border for each block tagged fact. Hovering over a tagged\r\n fact will highlight (shade) all content related to the tagged\r\n fact, and clicking on a tagged fact will reveal its tagging\r\n details in the Fact Review Window. Navigation, search and filter\r\n options are also provided to easily refine and identify specific\r\n types of tagged information.

      \r\n
      \r\n
      \r\n
      \r\n
      \r\n
      \r\n \r\n
      \r\n
      \r\n
      \r\n

      The Fact Review Window shows the tagging\r\n details for the currently selected fact, which is highlighted\r\n with a solid blue border. There are four categories of fact detail\r\n which can be viewed; an \"N/A\" value indicates there is no\r\n available information for the item within the given category:

      \r\n
        \r\n
      • Attributes - All primary information (as\r\n applicable) describing the tagged fact including period, sign,\r\n decimals, dimensional detail (axes and members), scale, measure\r\n and data type.
        • \r\n
      • Labels - Detailed documentation\r\n (definition) for the tag used, and other labels.
        • \r\n
      • References - Authoritative reference\r\n information (as applicable) for the selected tag.
        • \r\n
      • Calculation - Balance and calculation\r\n hierarchy details (numeric non-dimensioned items only).
        • \r\n
      \r\n
      \r\n
      \r\n
      \r\n
      \r\n
      \r\n \r\n
      \r\n
      \r\n
      \r\n

      There are two ways to search the\r\n document: FACTS AND SECTIONS. To search information, enter a\r\n keyword in a search box and select the magnifying glass icon to\r\n return matching results. Search operators \"and\" (via \"AND\" or\r\n \"&\") and \"or\" (via \"OR\" or \"|\") are available to further refine a\r\n search.

      \r\n

      \r\n FACTS\r\n

      \r\n

      The Search Facts box can be used to find\r\n tagged facts matching entered keywords. By default, tag name, tag\r\n labels, and tagged content are included in Search. Tagged facts\r\n matching the search criteria are shown with a yellow-colored\r\n (default) shading on the page and appear in the Facts List.\r\n Select the \"X\" button to the left of the magnifying glass icon to\r\n clear the search.

      \r\n

      The content included in Search can be increased to\r\n included tag definitions, dimensions, and authoritative\r\n references by selecting the cog / wheel icon to the left of the\r\n search box. See Settings for more information.

      \r\n\r\n

      \r\n SECTIONS\r\n

      \r\n

      The Search Sections box can be used to\r\n filter the tagged sections of the financial statements.

      \r\n
      \r\n
      \r\n
      \r\n
      \r\n
      \r\n \r\n
      \r\n
      \r\n
      \r\n

      Filters change the number of highlighted\r\n facts and items in the fact list by providing several ways to\r\n review and pinpoint certain tagged information. Multiple filters\r\n can be used at once.

      \r\n

      \r\n DATA FILTERS\r\n

      \r\n

      These filter options allow the user to\r\n refine the highlighted tagged facts by data type:

      \r\n
        \r\n
      • All - Displays all tagged data (default)
        • \r\n
      • Amounts Only - Numeric facts only
        • \r\n
      • Text Only - Textual facts only
        • \r\n
      • Calculations Only - Numeric facts\r\n participating in a calculation only
        • \r\n
      • Negatives Only - Negative Numeric amounts\r\n only
        • \r\n
      • Additional Items Only - Tagged facts\r\n without a corresponding HTML presentation (i.e., hidden facts)\r\n only
        • \r\n
      \r\n

      \r\n TAGS FILTER\r\n

      \r\n

      These filters allow the user to refine\r\n the highlighted facts by tag type:

      \r\n
        \r\n
      • All - Displays all tagged data (default)
        • \r\n
      • Standard Only- Facts with standard tags\r\n from a common fact (e.g., US_GAAP, IFRS, DEI).
        • \r\n
      • Custom Only - Facts with extension\r\n \"custom\" tags unique to the entity's document.
        • \r\n
      \r\n

      \r\n MORE FILTERS\r\n

      \r\n

      Additional filters that allow user to\r\n further refine the highlighted facts:

      \r\n
        \r\n
      • Periods - List of all used context years\r\n and reporting periods
        • \r\n
      • Measures - List of all used units of\r\n measure (e.g., US Dollars); as applicable
        • \r\n
      • Axis - List of all used axis\r\n (dimensions); as applicable (dimensions); as applicable
        • \r\n
      • Scale - List of all used scaled options\r\n (e.g., thousands, millions); as applicable
        • \r\n
      • Balance - Debit, credit; as applicable
        • \r\n
      \r\n

      Multiple filters work in conjunction\r\n with each other. For example, selecting the \"Amounts Only\" Data\r\n filter and \"Custom Only\" Tags filter will highlight only numeric\r\n tagged facts using custom tags and list them in the fact list.\r\n Active filters are displayed in the Filter toolbar as they are\r\n selected. Active filters can be removed individually by\r\n unchecking or selecting ALL option for each filter, or all at\r\n once via the \"Reset All Filters\" option.

      \r\n
      \r\n
      \r\n
      \r\n
      \r\n
      \r\n \r\n
      \r\n
      \r\n
      \r\n

      On the toolbar, select the \"count\" to\r\n the right of the word Facts (e.g., Facts 234) to reveal the Facts\r\n List results - a navigable listing of all currently highlighted\r\n tagged facts. The count represents the current number of facts.

      \r\n

      By default, all tagged facts are\r\n displayed in the Fact List Results. The list content and count\r\n reflects the currently highlighted facts (i.e., both Filters and\r\n Search criteria refine the list to match the highlighted tagged\r\n facts). Navigation controls are available to move through the\r\n list as well as move the current view to the corresponding\r\n highlighted fact location automatically. When a fact in the Facts\r\n Results List is selected, it will reveal the Fact Review Window.

      \r\n

      Certain letters may appear to the right\r\n of an item in the Fact Result list to indicate a certain\r\n property. If the letter \"A\" appears for a fact, it indicates the\r\n fact is additional data (i.e., hidden with potentially no\r\n corresponding HTML presentation). If the letter \"C\" appears, the\r\n fact is tagged with a custom tag. If the letter \"D\" appears, the\r\n fact is tagged with dimensional information.

      \r\n
      \r\n
      \r\n
      \r\n
      \r\n
      \r\n \r\n
      \r\n
      \r\n
      \r\n

      The SECTIONS toolbar item provides a\r\n listing of the tagged sections of the Inline document. Sections\r\n are divided into three groups:

      \r\n
        \r\n
      • Document & Entity Information
        • \r\n
      • Financial Statements
        • \r\n
      • Notes to the Financial Statements
        • \r\n
      \r\n

      By expanding a group and selecting a\r\n section item in the listing, Inline XBRL viewer will navigate to that section.\r\n When the Tagged Sections feature is open, the Search Sections box\r\n will additionally filter the list of sections to only those\r\n sections that match the entered criteria.

      \r\n
      \r\n
      \r\n
      \r\n
      \r\n
      \r\n \r\n
      \r\n
      \r\n
      \r\n

      \r\n Information\r\n

      \r\n

      The Information menu item provides\r\n additional detail about the current Inline document and\r\n customizable viewer settings.

      \r\n
        \r\n
      • Company and Document - Basic company and document\r\n information
        • \r\n
      • Tags - Fact and tag (standard and custom)\r\n information
        • \r\n
      • Files - Files used
        • \r\n
      • Additional Items - Additional hidden data\r\n that's been tagged but potentially does not have a corresponding\r\n location in the HTML
        • \r\n
      \r\n

      \r\n Save XBRL Instance\r\n

      \r\n

      The Save XBRL Instance menu item allows\r\n an XBRL instance document (*.xml) that is extracted from the\r\n Inline document to be saved locally for use by software\r\n supporting the XBRL format.

      \r\n

      \r\n Save XBRL Zip\r\n

      \r\n

      The Save XBRL Zip menu item allows a zip\r\n file (*.zip) that contains the as-provided XBRL instance document\r\n and related custom fact files to be saved locally.

      \r\n

      \r\n Settings\r\n

      \r\n

      The Settings menu item provides the\r\n ability to customize Viewer features.

      \r\n

      \r\n Tagged Fact Hover\r\n

      \r\n
        \r\n
      • On - Displays the hover fact review\r\n window for any tagged fact*
        • \r\n
      • Off - Hides the hover fact review window\r\n for any tagged fact (default)
        • \r\n
      \r\n

      * May impact performance with certain\r\n web browsers.

      \r\n

      \r\n Auto Scrolling Position (This setting will have\r\n no effect on IE 10 or Safari)\r\n

      \r\n
        \r\n
      • Top - Selecting a fact from the Sections\r\n Menu or the Fact Menu will automatically scroll that element to\r\n the Top of the Inline XBRL viewer window (default)
        • \r\n
      • Center - Selecting a fact from the\r\n Sections Menu or the Fact Menu will automatically scroll that\r\n element to the middle of the Inline XBRL viewer window
        • \r\n
      \r\n

      \r\n Highlight Colors (Use the save and reset buttons\r\n to save the selected setting or reset to default)\r\n

      \r\n
        \r\n
      • Tagged Data - Change the highlight color\r\n of the tagged fact border
        • \r\n
      • Search Results - Change the background\r\n color of tagged items matching the Search results
        • \r\n
      • Selected Fact - Change the color of\r\n highlight border used to identify the currently selected fact
        • \r\n
      • Tag Shading (hover) - Change the color of\r\n the shading applied to tagged data on hover
        • \r\n
      \r\n

      \r\n Search Options\r\n

      \r\n
        \r\n \r\n
      • Include Labels - Extends Search to\r\n include tag labels
        • \r\n
      • Include Definitions - Extends Search to\r\n include tag definitions
        • \r\n
      • Include Dimensions - Extends Search to\r\n include dimensional detail
        • \r\n
      • Include References - Extends Search to\r\n include authoritative reference information. Optionally specify\r\n the reference parts to include in the search (e.g., topic,\r\n subtopic, section)
        • \r\n
